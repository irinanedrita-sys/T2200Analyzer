# Contributing to T2200Analyzer

Thank you for considering contributing!

## How to contribute
1. Fork the repository.
2. Create a new branch for your changes.
3. Submit a pull request with a clear explanation of what was improved.

## Code style
- Use Python 3.10+
- Follow PEP8 formatting where reasonable.
- Add comments when introducing complex logic.

## Bug reports
If submitting a bug, please include:
- Example PDF (with personal details removed)
- Screenshot of the issue
- Error message, if any

## Feature suggestions
Feel free to open an Issue describing:
- The problem
- Why it would help accountants
- Proposed solution
